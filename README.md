# <img src="Icon.png" width="45" alt="App Icon" /> Precision Detector for Bee Swarm Simulator

A draggable AHK macro window that displays the time remaining on your precision buff and alerts you when it's time to refresh it in Bee Swarm Simulator.

![AHK](https://img.shields.io/badge/.AHK-AutoHotkey-green)

---

## Custom Warning Sound

If you don't like the default .mp3 alert, swap it out with any sound you prefer:

1. Delete the existing `warn.mp3` file from the folder
2. Add your own audio file to the same folder
3. Rename it to exactly `warn.mp3`

> The replacement file **must** use the exact same filename and extension: `warn.mp3`
